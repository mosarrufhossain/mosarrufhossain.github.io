# Mosarruf Hossain - GitHub Pages Version

This is a static multi-page website prepared to mirror the public Google Sites version of the site as closely as possible for GitHub Pages.

## Files
- `index.html`
- `about.html`
- `education.html`
- `research.html`
- `activities.html`
- `blog.html`
- `styles.css`
- `script.js`

## Publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save.
